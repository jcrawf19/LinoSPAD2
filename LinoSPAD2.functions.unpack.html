LinoSPAD2.functions.unpack module

Module with scripts for unpacking data from LinoSPAD2.

This file can also be imported as a module and contains the following functions:

  • unpack_bin - function for unpacking data from LinoSPAD2,

firmware version 2212. Utilizes the numpy library to speed up the process.

LinoSPAD2.functions.unpack.unpack_bin(file, board_number: str, timestamps: int = 512)[source]

Unpack data from firmware version 2212.

Unpacks binary-encoded data from LinoSPAD2 firmware version 2212. Uses numpy to achieve the best speed for unpacking. Data is returned as a 3D array where rows are TDC numbers, columns are the data, and each cell contains pixel number in the TDC (from 0 to 3) and the timestamp recorded by that pixel.

Parameters:

  • file (str) – ‘.dat’ data file.

  • board_number (str) – LinoSPAD2 daughterboard number. Either ‘A5’ or ‘NL11’ are recognized.

  • timestamps (int, optional) – Number of timestamps per TDC per acquisition cycle. The default is 512.

Raises:

TypeError – Controller for the type of ‘board_number’ parameter which should be a string. FileNotFoundError Controller for stopping the script in the case no calibration data were found.

Returns:

data_all – 3D array of pixel coordinates in the TDC and the timestamps.

Return type:

array-like
